#system-design #reference

# System Design Acronyms

| Acronym | Full Name | Quick Definition |
|---------|-----------|-----------------|
| **ACID** | Atomicity, Consistency, Isolation, Durability | Database transaction guarantees. [[01_fundamentals/acid_vs_base]] |
| **API** | Application Programming Interface | Contract for software communication. [[01_fundamentals/api_design]] |
| **BASE** | Basically Available, Soft state, Eventually consistent | Relaxed DB guarantees. [[01_fundamentals/acid_vs_base]] |
| **CAP** | Consistency, Availability, Partition tolerance | Distributed systems theorem. [[01_fundamentals/cap_theorem]] |
| **CDN** | Content Delivery Network | Serve content from edge locations. [[02_building_blocks/cdn]] |
| **CQRS** | Command Query Responsibility Segregation | Separate read/write models. [[03_design_patterns/cqrs]] |
| **CRUD** | Create, Read, Update, Delete | Basic data operations |
| **DAU** | Daily Active Users | Key product metric |
| **DLQ** | Dead Letter Queue | Queue for failed messages. [[02_building_blocks/message_queues]] |
| **DNS** | Domain Name System | Domain → IP resolution. [[01_fundamentals/networking_basics]] |
| **ELK** | Elasticsearch, Logstash, Kibana | Log management stack. [[02_building_blocks/monitoring_and_logging]] |
| **gRPC** | Google Remote Procedure Call | Binary RPC protocol. [[01_fundamentals/api_design]] |
| **HLS** | HTTP Live Streaming | Video streaming protocol. [[05_case_studies/design_video_streaming]] |
| **HTTP** | Hypertext Transfer Protocol | Web communication protocol. [[01_fundamentals/networking_basics]] |
| **LB** | Load Balancer | Distribute traffic. [[02_building_blocks/load_balancers]] |
| **LRU** | Least Recently Used | Cache eviction policy. [[02_building_blocks/caching]] |
| **LSM** | Log-Structured Merge Tree | Write-optimized storage. [[03_design_patterns/write_ahead_log]] |
| **QPS** | Queries Per Second | Throughput metric |
| **REST** | Representational State Transfer | API style. [[01_fundamentals/api_design]] |
| **RPC** | Remote Procedure Call | Call function on remote server |
| **SLA** | Service Level Agreement | Uptime contract. [[02_building_blocks/monitoring_and_logging]] |
| **SLI** | Service Level Indicator | What you measure |
| **SLO** | Service Level Objective | What you target |
| **SPOF** | Single Point of Failure | Component whose failure kills the system |
| **SQL** | Structured Query Language | Relational DB language. [[02_building_blocks/databases_sql]] |
| **SSE** | Server-Sent Events | Server push over HTTP |
| **TCP** | Transmission Control Protocol | Reliable delivery. [[01_fundamentals/networking_basics]] |
| **TLS** | Transport Layer Security | Encryption protocol |
| **TTL** | Time To Live | Expiration timer (cache, DNS) |
| **UDP** | User Datagram Protocol | Fast, unreliable delivery. [[01_fundamentals/networking_basics]] |
| **WAL** | Write-Ahead Log | Durability mechanism. [[03_design_patterns/write_ahead_log]] |
